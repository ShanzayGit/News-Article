# Newspaper Article Project

## Overview

This project is a **Newspaper Article webpage** created using **HTML and CSS**. It recreates a traditional newspaper-style layout featuring a publication title, date, headline, author section, and formatted article content.

The design focuses on typography, spacing, readability, and visual hierarchy to provide an authentic newspaper reading experience.

## Features

* Newspaper-inspired layout design
* Semantic HTML structure
* Responsive centered article container
* Styled headings and typography
* Article formatting with indentation
* First-letter emphasis effect
* Box shadow and border styling for visual depth

## Technologies Used

* HTML5
* CSS3

* Uppercase styling for publication name and author
* Italic sub-headline styling
* Paragraph indentation for article appearance
